# 2025FPGA AMD自主命题赛道

## 📖 项目简介
本项目为 **2025 年全国大学生 FPGA 创新设计大赛（AMD 自主命题赛道）** 参赛作品。  
项目基于 **Xilinx Zynq SoC 平台** 实现，采用 **Vivado2024.2 + Vitis** 工具链进行硬件与软件协同开发。  

主要实现：
-  **FPGA 在通信信号调制解调中的高效实现方案**  
-  **AM / FM / PSK 调制信号的实时检测与分析**  
- 提供可视化的波形测量与参数输出

---

## ⚙️ 硬件平台
- **FPGA 芯片：** AMD Xilinx Zynq-7020 系列  
- **ADC/DAC ：** 3PA1030/DAC904
